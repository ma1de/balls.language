# syntax: 
">" - forward by 1 <br/>
"<" - backward by 1 <br/>
";" - forward by 0.5 <br/>
":" - backward by 0.5 <br/>
"^" - the whole number to the power of n, where n is equal to the amount of the ^ symbols <br/>
"[" - square root of the whole number <br/>
"1" - digit
